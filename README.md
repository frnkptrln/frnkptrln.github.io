# frnkptrln.github.io

Personal repository index and homepage.  
Hosted via GitHub Pages at [frnkptrln.github.io](https://frnkptrln.github.io).

## Overview
This repository serves as a static, dependency-free landing page for public projects, essays, and systems-related research. It is built with a strict minimalist aesthetic.

## Stack
- **HTML5**: Pure semantic markup.
- **CSS3**: Vanilla CSS, no frameworks, brutalist design principles.
- **JavaScript**: Minimal vanilla JS for subtle interactions.

## Local Development
To preview the site locally, run any basic HTTP server from the root directory:

```bash
python3 -m http.server 8080
```
Then open `http://localhost:8080` in your browser.

*Note: This index is a partial surface representation. Much of the underlying work, experiments, and structures exist beyond this space — operating offline, conceptually, or within undocumented contexts.*
