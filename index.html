Defensive Threat Intelligence and OSINT toolkit for authorized analysis of modern leak-site surfaces. The first crawler handles Tor-routed SPA ?path= listings, JWT authentication, local indicator scanning, and reviewable follow-up commands without automatic downloads.

the-weaving-sound SuperCollider Audio

Generative sound synthesis and algorithmic composition. The main work is an infinite SuperCollider installation driven by a five-phase conductor, with standalone sketches ranging from dub techno to unstable generative studies.

-> Main Work -> Sketches
the-weaving-void Python Esolang

Archive of esoteric programming languages with Python interpreters and documentation. Distaff, Vortex, MandelMemory, SFract, Entanglement, Gastronomy, Clockwork, Bloom, and Marginalia explore computation as constraint, ritual, sound, cooking, growth, annotated memory, and verification.

ble-density-gauge C Flipper Zero Prototype

Flipper Zero BLE density gauge concept with an analog System Mass Index interface. Currently a technical UI prototype because external `.fap` apps cannot access the low-level passive BLE radio commands required for real scanning.

flipper-zero-pinball C Flipper Zero

Portrait-mode pinball game for the Flipper Zero monochrome display with flippers, bumpers, launcher charging, scoring, and sound effects.

arch-pacman-clicker HTML

Dependency-free browser clicker styled like an Arch terminal. Build packages, buy automation upgrades, and optimize package throughput.
